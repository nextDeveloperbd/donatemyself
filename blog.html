Important FAQ

What is the Document Object Model (DOM)?

Web documents can be programmed using the Document Object Model (DOM). Using computer languages like JavaScript, developers can dynamically interact with and modify a webpage's content, structure, and styles by representing the webpage's structure as a tree of objects.

What is Event delegation in the context of the dom, and why is it useful?

JavaScript's event delegation technique involves attaching a single event listener to a parent element so that it can handle events for its child elements. The event listener on the parent element captures events as they "bubble up" from the child elements rather than adding separate event listeners to each of the child elements. Event bubbling, a phenomenon whereby events initiated on a child element spread to its ancestors, is the reason this works.

How do you manipulate an element's attribute and style using the DOM ?

Use getAttribute(), setAttribute(), and removeAttribute() to modify attributes, or use properties like id directly. Use classList to manage classes or the style property to change styles.toggle(), add(), and remove(). To access computed styles, use getComputedStyle().
